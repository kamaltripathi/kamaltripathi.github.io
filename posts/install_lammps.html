Installing LAMMPS on Linux

This guide shows how to install LAMMPS from source on a Linux system, including optional packages.

Uninstall any existing version

sudo apt-get remove --auto-remove lammps

Download and extract the source

wget https://www.lammps.org/download.html
tar -xzvf lammps*.tar.gz

Build LAMMPS

cd lammps/src
make yes-MC
make yes-KSPACE
make yes-rigid
make yes-misc
make yes-basic
make yes-molecule
make yes-dipole
make yes-EXTRA-PAIR

Check which packages are included:

make ps

Build serial or MPI version:

make serial
make mpi

Install binaries

After building, copy the executables to /bin so they are available globally:

sudo cp lmp_serial /bin
sudo cp lmp_mpi /bin

Adding additional packages

If you want to include extra packages later:

make yes-packagename
make serial
make mpi
