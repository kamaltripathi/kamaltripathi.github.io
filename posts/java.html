Installing LAMMPS on Linux

Published on 2025-08-14

In this post, I will explain how to install LAMMPS, a widely used molecular dynamics simulator, on a Linux system. These instructions assume Ubuntu/Debian, but are adaptable to other distributions.

1. Update your system

sudo apt update
sudo apt upgrade

2. Install dependencies

sudo apt install build-essential cmake git fftw3-dev libjpeg-dev libpng-dev libblas-dev liblapack-dev

3. Clone the LAMMPS repository

git clone https://github.com/lammps/lammps.git
cd lammps

4. Build LAMMPS

mkdir build
cd build
cmake ../cmake
make -j4
sudo make install

5. Verify installation

lmp -h

If LAMMPS runs and shows the help message, the installation was successful!

Notes:

  • You can also install via conda using conda install -c conda-forge lammps.
  • For GPU acceleration, additional packages like CUDA are required.
