Git Cheatsheet

Git is a version control system that helps manage projects efficiently. Here’s a quick reference for common Git commands.

Initial Setup

git config --global user.name "username"
git config --global user.email "useremail@example.com"
git config    # Check configuration

Essential Commands

cd path/to/directory/
git init           # Initialize a new repository
git clone url      # Clone repository from GitHub
git add filename   # Add file to staging area
git add *          # Add all files to staging
git commit -m "msg"  # Commit changes
git push origin master
git push origin main

Frequently Used Commands

git status
git remote add origin url-to-repo.git

Other Useful Commands

git rm filename              # Remove from staging and working directory
git rm --cached filename       # Remove only from staging
git reset HEAD filename        # Unstage a file
git reset HEAD .               # Unstage all files
.gitignore                     # Ignore files/folders
git branch branchname          # Create branch
git checkout branchname        # Switch branch
git config --get remote.origin.url  # Show remote URL

For more details, see Atlassian Git Glossary.
