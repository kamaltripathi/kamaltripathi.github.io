Git Cheatsheet

A quick reference for common Git commands used to manage projects and website updates efficiently.

Initial Setup & Identity

git config --global user.name "Kamal Tripathi"
git config --global user.email "your-email@example.com"
git config --global credential.helper store  # Remembers your Token
git config --list                            # Check configuration

Essential Workflow (Save & Upload)

git status                         # See modified files
git add .                          # Stage all changes
git commit -m "Your message here"  # Commit changes
git push origin main               # Upload to GitHub (main branch)

Branching & Maintenance

git branch                         # List local branches
git branch -m master main          # Rename master to main
git checkout -b branch-name        # Create and switch to new branch
git fetch --prune                  # Remove old remote references

Remote & Repository Setup

git init                           # Initialize new local repo
git clone https://url-to-repo.git  # Clone existing repo
git remote add origin url-to-repo  # Link to GitHub
git remote -v                      # Show remote URL

Reverting & Unstaging

git checkout filename              # Undo changes to a specific file
git reset HEAD filename            # Unstage a file
git reset --hard HEAD              # Wipe all uncommitted changes

For more details, see Atlassian Git Glossary.
