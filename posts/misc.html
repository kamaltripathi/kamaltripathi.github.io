Useful Linux Terminal Snippets

A collection of commands for file organization, process management, and media processing.

Organize Files into Folders Automatically

Create folders based on file names and move files into them (useful for .data files):

find . -name "*.data" -exec sh -c 'mkdir "${1%.*}"; mv "$1" "${1%.*}"' _ {} \;

Find and delete files

find . -name "slurm-*" -exec rm -r {} \;

Run programs in background (nohup)

Keep a process running after closing the terminal:

nohup bash afterrun.sh > output_new.txt 2>&1 &

Create video from images (FFmpeg)

ffmpeg -framerate 1 -pattern_type glob -i '*.jpg' -c:v libx264 -r 30 -pix_fmt yuv420p out.mp4

Splitting PDF files

qpdf --empty --pages infile.pdf 1-5 -- outfile.pdf

Stitching JPGs to make a GIF

convert -delay 10 -loop 4 snap.*.rgb movie.gif
