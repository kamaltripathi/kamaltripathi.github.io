Cluster Access and File Transfer

Instructions for logging into 3SR and Gricad clusters, transferring files, and running simulations.

Login and File Transfer — 3SR Cluster

login 3sr gate
ssh ktripathi@gate.3sr-grenoble.fr
ssh ktripathi@goliarda or ssh goliarda
scp -r /home/kamalt/Desktop/file.py ktripathi@gate.3sr-grenoble.fr:/home/3S-LAB/ktripathi/
# from gate.3sr, one can scp again
scp -r sphere_equi.pdf goliarda:/home/3S-LAB/ktripathi/

# Single command scp from laptop to goliarda
scp -r -o ProxyCommand="ssh ktripathi@gate.3sr-grenoble.fr nc goliarda 22" /home/kamalt/Desktop/sphere_equi.pdf ktripathi@goliarda:/home/3S-LAB/ktripathi/

pwd gives home/3S-LAB/ktripathi

Running Simulations on Goliarda

1. Create a gate in a terminal:
ssh -L 2222:goliarda:22 ktripathi@gate.3sr-grenoble.fr

2. Open another terminal:
ssh -p 2222 ktripathi@localhost
# Now you're connected

3. Transfer files to cluster:
scp -rP 2222 test-cluster ktripathi@localhost:/home/3S-LAB/ktripathi/

4. Collect data:
scp '-P 2222' file ktripathi@localhost:/home/3S-LAB/ktripathi
scp -r -o ProxyCommand="ssh ktripathi@gate.3sr-grenoble.fr nc goliarda 22" ktripathi@goliarda:/home/3S-LAB/ktripathi/FoldertoGet ./media/kamalt/SystemHDD/Research/UGA/

# From 3SR Laptop to my Laptop
scp -r -o ProxyCommand="ssh ktripathi@gate.3sr-grenoble.fr nc lt-com056 22" ktripathi@lt-com056:/home/3S-LAB/ktripathi/Figure.pdf ./

# From My Laptop to 3sr Laptop

Goliarda Commands
