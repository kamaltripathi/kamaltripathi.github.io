Cluster Cheatsheet for IMSc

This is a place where I keep all the commands I commonly use while working on the IMSc cluster “Nandadevi”. It includes commands for job submission, monitoring, and managing resources.

Initial Setup

ssh username@clustername   # e.g., clustername = nandadevi
cd /path/to/directory       # e.g., /lustre/myusername

Essential Commands

sbatch slurm.sh        # submit a job
sinfo                  # cluster information
qstat -u username      # running jobs
qstat -u username | grep -i "nandasq"   # jobs on specific cluster
qstat -u username | grep -i "nandasq" | wc -l  # number of jobs
qstat -u username | grep -i "nandasq" | awk '{sum+=$7} END {print sum}'  # total CPUs used

Note: Maximum allowed CPUs is 96.

Frequently Used Commands

scancel jobid   # cancel a job
